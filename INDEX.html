Web Technology

HTML waxay u taagan tahay HyperText Markup Luuqad Waxaa loo isticmaalaa in lagu naqshadeeyo boggaga internetka oo isticmaalaya luqad calaamadayn ah. HTML waa isku dhafka Hypertext iyo luqadda Markup. Hypertext qeexaya isku xirka shabakada bogag Luuqad calaamadeyn ah ayaa la adeegsadaa si loo qeexo qoraalka qoraalka dhexdiisa tag kaas oo qeexaya qaab-dhismeedka bogagga shabakadda. HTML waa luuqad calaamadeysan kaas oo loo isticmaalo browser-ku wax ka beddel qoraalka, sawirada, iyo kuwa kale ka kooban si ay u muujiyaan in loo baahan yahay qaab.

C Programming

C waa luqadda barnaamijka. Markii hore waxaa soo saaray Dennis Ritchie sida barnaamij habaysan luqadda lagu qoro nidaamka hawlgalka. Tilmaamaha ugu muhiimsan ee luqadda C waxaa ka mid ah heerka hoose ee helitaanka xusuusta, set fudud ereyada muhiimka ah, iyo qaab nadiif ah, kuwan Tilmaamo ka dhigaya luqadda C ku habboon barnaamijyada nidaamka sida nidaamka hawlgalka ama horumarinta isku-darka.

Java

Java waa mid ka mid ah kuwa ugu caansan luqadda barnaamijyada caanka ah sanado badan. Java waa Shay Hanuunsan. Si kastaba ha ahaatee ma aha loo tixgaliyey sidii shayga saafiga ah u janjeedha sida ay u bixiso taageerada asaasiga ah noocyada xogta (sida int, char, iwm) The Koodhadhka Java ayaa marka hore lagu soo ururiyey byte code (mashiinka madax banaan code). Markaa code byte waxa uu ku shaqeeya Java Virtual Mashiinka (JVM) iyada oo aan loo eegin dhismaha hoose.
